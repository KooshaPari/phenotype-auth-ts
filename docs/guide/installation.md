# Installation

## Requirements

- Node.js 18+
- npm or yarn or pnpm

## Package Installation

```bash
npm install @phenotype/auth-ts
```

## Development Setup

Clone the repository:

```bash
git clone https://github.com/KooshaPari/phenotype-auth-ts.git
cd phenotype-auth-ts
```

Install dependencies:

```bash
npm install
```

## TypeScript Configuration

Add to your `tsconfig.json`:

```json
{
  "compilerOptions": {
    "module": "ESNext",
    "moduleResolution": "bundler",
    "strict": true
  }
}
```

## Verification

Run the test suite to verify installation:

```bash
npm test
```

Build the project:

```bash
npm run build
```
